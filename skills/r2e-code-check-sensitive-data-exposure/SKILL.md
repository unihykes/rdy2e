## 指令(做什么)

- 扫描用户指定位置的代码文件。
- 检查是否存在敏感数据暴露风险。
- 给出修复建议（最小暴露、脱敏、避免硬编码、限制日志与对外输出等）。

## 约束(怎么做)

- [范围] C++ 文件（h/hpp/cpp）
- [命中] 敏感信息（如密钥、令牌、口令、个人隐私字段）以明文写入日志、异常、响应或可外泄通道。
- [命中] 敏感配置或凭据被硬编码，或未脱敏直接透传。
- [排除] 已按规范进行掩码/脱敏/加密且不暴露原值的场景不标记。
